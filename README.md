\# Life Simulator



Life Simulator is a Flutter-based simulation app that models daily life decisions and tracks stats such as health, productivity, and progress over time.



\## Features

\- Day-by-day life simulation logic

\- Dynamic stat tracking system

\- Decision-based outcomes

\- Clean dashboard-style UI

\- APK generated and tested on real devices



\## Tech Stack

\- Flutter

\- Dart

\- Android SDK



\## What I Learned

\- State management in Flutter

\- Designing reusable widgets

\- Logic implementation for simulations

\- Debugging runtime and UI issues
